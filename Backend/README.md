# Backend - Prueba Técnica

Backend desarrollado como parte de una prueba técnica, implementando una API REST escalable, estructurada y preparada para producción.

## Tecnologías utilizadas

- Python
- Flask
- SQLAlchemy
- Alembic (migraciones de base de datos)
- SQLite / PostgreSQL / MySQL
- dotenv (variables de entorno)

## Arquitectura

El proyecto sigue una arquitectura modular basada en:

- Separación de responsabilidades
- Controladores (routes)
- Servicios
- Modelos
- Configuración centralizada

Esto permite escalabilidad, mantenibilidad y facilidad de pruebas.

## Requisitos previos

Antes de ejecutar el proyecto debes tener instalado:

- Python 3.10 o superior
- pip
- virtualenv (recomendado)
